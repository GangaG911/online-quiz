# online-quiz# 🧠 Online Quiz App

A simple and interactive **Online Quiz Application** built using **HTML, CSS, and JavaScript**.
This project is beginner-friendly and helps understand basic DOM manipulation and logic building.

---

## 🚀 Features

* 📌 Multiple choice questions
* ✅ Instant answer checking
* 🎯 Score calculation
* 🎉 Final result display
* 💡 Simple and clean UI

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
Online-Quiz-App/
│── index.html
│── README.md
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Double click on `index.html`
4. The quiz will open in your browser 🎉
